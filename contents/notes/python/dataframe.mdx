### drop_duplicates
```python
df = pd.DataFrame(columns=['A', 'B', 'C'], data=[[1,2,3],[1,2,3],[4,5,6]])
	A	B	C
0	1	2	3
1	1	2	3
2	4	5	6

df.drop_duplicates()
	A	B	C
0	1	2	3
2	4	5	6
```
