## 句型
- 私(わたし)はXXXです 我是XXX
- AはBではありません A不是B
- AはBですか A是B嗎
- AもBです A也是B
- これはAです 這是A
	- これ 這個 / それ 那個 / あれ 那個(遠的) / どれ 哪個
- これはAのBです 這是A的B
- このＢはＡのです 這個B是A的
- これは誰(だれ)のですか 這是誰的
- これは何(なに)ですか 這是什麼
- ここは＿です 這裡是＿
  - ここは教室(きょうしつ)です 這裡是教室
	- 図書館(としょかん) 圖書館
	- 食堂 しょくどう
	- トイレ 廁所
  - 場所 ここ / そこ / あそこ / どこ
  - 場所(有禮貌) こちら / そちら / あちら / どちら
- (地方/人物/東西)はあそこです XXX在那裡
- (地方/人物/東西)はどこですか XXX在哪裡
