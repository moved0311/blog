|           |                |                                                            |
| --------- | -------------- | ---------------------------------------------------------- |
| 有         | あります 沒有生命(植物)  | 車があります                                                     |
| 在         |                | 東京ディズニーランドは 千葉県に あります                                      |
|           | います 有生命        | ロビーに 佐藤さんが います<br/>家族は ニョーヨークに います 我的家人在紐約                 |
| 正在        | て形 + います       | [[grammar#Lesson 14\|Lesson 14]]                           |
| 目前狀態      | て形 + います       | けんさんは 東京に住んでいます                                            |
| 我有        |                | 車を 持っています                                                  |
| 邀請邀約      | ませんか           | 一緒に 神戶へ 行きませんか [[grammar#Lesson 6\|Lesson 6]]              |
|           | ましょう           | 休みましょう 休息一下吧                                               |
| 請給我、請做    | ください           | 水をください                                                     |
| 請做        | て形+ください        | ドアを開けてください [[grammar#Lesson 14\|Lesson 14]]                |
| 可以做某事嗎？   | てもいいですか        | 荷物を預けてもいいですか                                               |
| 請做某事      | て形 + ください      | ここに書いてください 請寫在這裡<br/>少し待ってください請稍等片刻                        |
| 請不要做某事    | ないでください        | 辛くしないでください 請不要做辣的                                          |
| 可以...嗎？   | て形 + もいいですか    |                                                            |
| 想做某事(動作)  | ~~ます~~ + たいです  | 寿司を食べたいです [[grammar#Lesson 13\|Lesson 13]]                 |
| 去做        | ~~ます~~ + に行きます | 私は フランスへ 料理を 習いに 行きます<br/>[[grammar#Lesson 13\|Lesson 13]] |
| 要不要(為你)做  | ~~ます~~ + ましょうか | 荷物を 持ちましょうか 要不要幫你拿行李 ? [[grammar#Lesson 14\|Lesson 14]]    |
| 可以 / 許可表現 | て形 + もいいです     | 写真を 撮っても いいですか?<br/>飲み物を飲んでもいいです 可以喝東西                     |
| 不能/不可以    | て形 + はいけません    | お酒を 飲んでは いけません 不能喝酒                                        |
| 動作順序      | て形、て形、～ます      | 朝 ジョギングを して シャワーを 浴びて 会社へ 行きます 早上慢跑、洗澡然後去公司                |
| 做~之後      | て形から           | コンサートが 終わってから、レストランで 食事しました 音樂會結束後，在餐廳用餐                   |
| 既A又B      | Aて形、B          | この部屋は 広くて、明るいです 這個房間又寬敞又明亮                                 |
