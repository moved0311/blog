---
title: 助詞(じょし)
category: note
date: 2025-10-13
---

| 助詞      | 功能             | 例句                                 |
| ------- | -------------- | ---------------------------------- |
| は (讀作わ) | 表示句子主題(說明對象)   |                                    |
|         | 表示對比           |                                    |
|         | 表示特別提示         |                                    |
|         | 加在助詞後面表示強調     |                                    |
| の       | 表示所屬/所有關係      | 私の傘です                              |
|         | 內容             | それは日本の本です                          |
| に       | 時間             |                                    |
|         | 地點/到達目的地       | 博多(はかた)に行きます                       |
|         | 間接對象           |                                    |
| へ(讀作え)  | 往那個方向前進        |                                    |
| で       | 地點             | レストランで食べます 在餐廳吃飯                   |
|         | 手段/方法/工具       | カードで払(はら)います 用卡付款<br/>バスで行きます 搭公車去 |
|         | 所要數量           |                                    |
|         | 原因             |                                    |
|         | 限度             |                                    |
| から      | 表示起點(從)        |                                    |
| を       | 對象/受詞/動作作用對象   | ラーメンを食べます                          |
| か       | 表示疑問/反問/驚訝     |                                    |
| が       | 主詞/焦點/表示/前言/但是 |                                    |
| し       | 列舉理由           |                                    |
| など      | 列舉、等等          |                                    |
| しか      | 只              |                                    |
| と       | 提示內容           |                                    |
