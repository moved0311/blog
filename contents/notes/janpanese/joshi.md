---
title: 助詞(じょし)
category: note
date: 2025-10-13
---
- から 表示起點(從)
- を 對象/受詞/動作作用對象
- へ(讀作え) 方向(往)
- か 表示疑問/反問/驚訝
- が 主詞/焦點/表示/前言/但是
- し 列舉理由
- など 列舉、等等
- しか 只
- と 提示內容

| は (讀作わ) | 表示句子主題(說明對象) |                      |                |
| ------- | ------------ | -------------------- | -------------- |
|         | 表示對比         |                      |                |
|         | 表示特別提示       |                      |                |
|         | 加在助詞後面表示強調   |                      |                |
| の       | 表示所屬         |                      |                |
| に       | 時間           |                      |                |
|         | 方面           |                      |                |
|         | 間接對象         |                      |                |
| で       | 地點           | レストランで食べます           | 在餐廳吃飯          |
|         | 手段/方法/工具     | カードで払います<br/>バスで行きます | 用卡付款<br />搭公車去 |
|         | 所要數量         |                      |                |
|         | 原因           |                      |                |
|         | 限度           |                      |                |
