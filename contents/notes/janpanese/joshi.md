| --------- | ------------ | ------------------------------------------------------------------------ |
| は (讀作わ)   | 主題           | 私は日本人です                                                                  |

|           | 表示特別提示       |                                                                          |
|           | 加在助詞後面表示強調   |                                                                          |
| を         | 對象/受詞/動作作用對象 | ラーメンを食べます                                                                |
|           | 經過地點         | 公園を散歩します　[[grammar#Lesson 13\|Lesson 13]]<br/>橋を 渡(わた)ります                |
|           | 離脫點          | 喫茶店を出ます [[grammar#Lesson 14\|Lesson 14]]<br/>バスを降ります                     |
| の         | 表示所屬/所有關係    | 私の傘です                                                                    |
|           | 內容           | それは日本の本です                                                                |
| に         | 時間           | 毎朝六時に起きます                                                                |
|           | 間接對象         | 友達に会います 我見朋友                                                             |
|           | 動作對象         | 私は木村さんに花をおげます我送花給木村先生                                                    |
|           | 地點/到達目的地     | 博多(はかた)に行きます                                                             |
|           | 存在位置         | あそこにコンビニがおります                                                            |
|           | 頻率/分母        | 一週間に1回映画を見ます                                                             |
|           | 目的           | 私は フランスへ 料理を 習いに 行きます [[grammar#Lesson 13\|Lesson 13]]                   |
|           | 進入點          | 喫茶店に入ります [[grammar#Lesson 14\|Lesson 14]]                                |
| へ(讀作え)    | 往那個方向前進      | 私は 京都へ 行きます 我去京都 [[grammar#Lesson 5\|Lesson 5]]                          |
| で         | 地點           | レストランで 食べます 在餐廳吃飯                                                        |
|           | 手段/方法/工具     | カードで払(はら)います 用卡付款<br/>私はパソコンで映画を見ます用電腦看電影 [[grammar#Lesson 7\|Lesson 7]] |
|           | 交通工具         | バスで行きます 搭公車去<br/>私はバイクでうちへ                                               |
|           | 所要數量/動作進行單位  | 私は 一人で 学校へ 来ました 我一個人來學校                                                  |
|           | 範圍           | 私は 1年で 夏が いちばん 好きです 我一年中最喜歡夏天                                            |
|           | 原因           |                                                                          |
|           | 限度           |                                                                          |
| が         | 焦點           | 私はイタリア料理が 好きです                                                           |
|           | 述語的主體        | 雨が降ります<br/>このお弁当は妻が作りました                                                 |
|           | 表示/前言/但是     |                                                                          |
| から...まで   | 表示從起點...到終點  | 台北から台南まで                                                                 |
| から        | 動作授方         | 私は カリナさんから チョコレートを もらいました<br/>我從卡莉娜那裡得到了巧克力<br/>私は先生に(から) 辞書を借りました       |
| から        | 表示原因理由       | 今日は 子供の 誕生日ですから、早く 帰ります                                                  |
| か         | 表示疑問/反問/驚訝   |                                                                          |
|           | 表示不確定        | 冬休みは どこか 行きましたか。 寒假你去哪裡了?<br/>何か食べますか                                    |

| など        | 列舉、等等        |                                                                          |
| しか        | 只            |                                                                          |

| や...など    | 並列(多樣)/和     | [[grammar#Lesson 10\|Lesson 10]]                                         |
