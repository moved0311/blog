---
title: 助詞(じょし)
category: note
date: 2025-10-13
---

| 助詞      | 功能             | 例句                                              |
| ------- | -------------- | ----------------------------------------------- |
| は (讀作わ) | 表示句子主題(說明對象)   |                                                 |
|         | 表示對比           |                                                 |
|         | 表示特別提示         |                                                 |
|         | 加在助詞後面表示強調     |                                                 |
| の       | 表示所屬/所有關係      | 私の傘です                                           |
|         | 內容             | それは日本の本です                                       |
| に       | 時間             | 毎朝六時に起きます                                       |
|         | 地點/到達目的地       | 博多(はかた)に行きます                                    |
|         | 間接對象           | 友達に会います 我見朋友                                    |
| へ(讀作え)  | 往那個方向前進        | 私は 京都へ 行きます 我去京都 [[grammar#Lesson 5\|Lesson 5]] |
| で       | 地點             | レストランで 食べます 在餐廳吃飯                               |
|         | 手段/方法          | カードで払(はら)います 用卡付款                               |
|         | 交通工具           | バスで行きます 搭公車去<br/>私はバイクでうちへ                      |
|         | 所要數量/動作進行單位    | 私は 一人で 学校へ 来ました 我一個人來學校                         |
|         | 原因             |                                                 |
|         | 限度             |                                                 |
| から...まで | 表示從起點...到終點    | 台北から台南まで                                        |
| を       | 對象/受詞/動作作用對象   | ラーメンを食べます                                       |
| か       | 表示疑問/反問/驚訝     |                                                 |
| が       | 主詞/焦點/表示/前言/但是 |                                                 |
| し       | 列舉理由           |                                                 |
| など      | 列舉、等等          |                                                 |
| しか      | 只              |                                                 |
| と       | 和(並列)          | 土曜日と日曜日                                         |
|         | 跟              | 私は家族と日本へ来ました                                    |
|         | 提示內容           |                                                 |
