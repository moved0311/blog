日語的語序通常依照主詞、受詞、動詞的順序來排列。

## 文型
1. 肯定句
	- A は B です。 A是B
	- A も B です。 A也是B
	- A は B の C です。 A是B的C
2. 否定句
	 - A は B じゃありません
	 - A は B ではありません
3. 疑問句
	 - A は B ですか。
	 - はい (是) / いいえ (不是)

#### 指示詞
1. これは A です。 這是A(近自己)
2. それは A です。 那是A(近對方)
3. あれは A です。 那是A(遠方)
4. この本 は 私の です。 這本書是我的 
 - この + 名詞
5. ここ は 学校 です。 這裡是學校
	- ここ 這裡
	- そこ 那裡
	- あそこ 那裡(遠方)
	- どこ 哪裡
6. エレベーター は あそこです 電梯在那裡
7. これ は 日本 の 車です 這是日本的車
8. この ネクタイ は 1500円です 這條領帶1500日圓
   - この ネクタイ は いくらですか 這條領帶多少錢? 

#### 時間
1. わたしは $\overset{まいあさ}{毎朝}$ 6時に 起きます 我每天早上6點起床
  - に 助詞表示時間點
2. わたしは $\overset{きのう}{昨日}$ $\overset{べんきょう}{勉強}$しました。 我昨天有讀書
  - しました 過去式
	||肯定|否定|
	|--|--|--|
	|現在式|ます|ません|
	|過去式|ました|ませんでした|
3. 休み は 土曜日 と 日曜日 です。 休息日是星期六和星期日
  - と 助詞表示並列關係 (和)
4. 昼休み は 12時から 1時まで です。 午休是從12點到1點
	- から 助詞表示起點(從)
	- まで 助詞表示終點(到)

#### Lesson 5
1. わたし は 京都 へ 行きます 我去京都
	- へ 助詞表示方向(往) 助詞讀作え
2. わたしは タクシーで うちへ 帰ります 我搭計程車回家
	- で 助詞表示交通工具
3. わたしは 家族と 日本へ 来ました 我和家人來日本
	- と 助詞表示一起(和)
	- わたしは 一人で 学校へ 来ました 我一個人來學校
		- で 助詞表示動作進行單位
4. わたしは 7月15日に 国へ 帰ります 我7月15日回國
	- に 助詞表示時間點
5. どこ へも 行きませんでした 我哪裡都沒去
	- も 也/都 全否定
	
#### Lesson 6
1. わたしは 本を 読みます 我看書
	- を 助詞表示動作作用對象
	- わたしは 水を 飲みます 我喝水
2. わたしは 駅で 新聞を 買いました 我在車站買報紙
	- で 助詞表示動作發生地點
3. 一緒に 神戶へ 行きませんか 我們一起去神戶好嗎?
	- ません(否定型) + か 表示邀請
	- いっしょに 映画を 見ませんか 我們一起去看電影好嗎
    - ええ、見るしょう
		- すみません、ちょっと...
4. ちょっと 休みましょう 休息一下吧
  - ましょう 吧
	- 帰りましょう
5. うちで 宿題を します。それから、CDを 聞きます。我在家做作業。然後聽CD
	- それから 然後
  - ときどき 有時候
6. 友達 に 会います 我見朋友
	- に 助詞表示間接對象/接觸點
7. 何
  - 有兩種唸法なん / なに
		- た行,だ行,な行 唸なん
		- 其他唸なに

## 文法
- 用の連接兩個名詞
- い形容詞 + 名詞
	- おいしい + パン 美味的麵包
	- やすい + ホテル 便宜的旅館
- な形容詞 + な + 名詞
	- にぎやか な まち 熱鬧的街道
	- ゆうめい な ばしょ 有名的地方(有名是い結尾但是是な形容詞)

形容詞(い形容詞)變化規則:
- 形容詞い + 名詞
- 形容詞 ~い~ + くない (否定)
- 形容詞~い~ + くて (中止型,還沒講完)
- 形容詞~い~ + かった (過去) 

形容動詞(な形容詞)變化
- + だ 原形
- + で 表示還沒講完(既...又...)
- + な 後面加名詞
- + に 後面加動詞

## 助詞
### 格助詞
單字與動詞的關係，單字的意思是格助詞決定的
- で 地點
- から 表示起點(從)
- に 間接對象/時間
- を 對象/受詞
- が 主詞

### 係助詞/副助詞
- は (要唸成わ)
	- 表示句子主題(說明對象)
	- 表示對比
	- 表示特別提示
	- 加在助詞後面表示強調

### 助動詞
- ます/まし 表示禮貌
- ない 否定助動詞

### 終助詞
- か 表示疑問/反問/驚訝
