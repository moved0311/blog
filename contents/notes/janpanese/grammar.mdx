日語的語序通常依照主詞、受詞、動詞的順序來排列。

## 文型
1. 肯定句
	- A は B です。 A是B
	- A も B です。 A也是B
	- A は B の C です。 A是B的C
2. 否定句
	 - A は B じゃありません
	 - A は B ではありません
3. 疑問句
	 - A は B ですか。
	 - はい (是) / いいえ (不是)

#### 指示詞
1. これは A です。 這是A(近自己)
2. それは A です。 那是A(近對方)
3. あれは A です。 那是A(遠方)
4. この本 は 私の です。 這本書是我的 
 - この + 名詞
5. ここ は 学校 です。 這裡是學校
	- ここ 這裡
	- そこ 那裡
	- あそこ 那裡(遠方)
	- どこ 哪裡
6. エレベーター は あそこです 電梯在那裡
7. これ は 日本 の 車です 這是日本的車
8. この ネクタイ は 1500円です 這條領帶1500日圓
   - この ネクタイ は いくらですか 這條領帶多少錢? 

#### 時間
1. わたしは $\overset{まいあさ}{毎朝}$ 6時に 起きます 我每天早上6點起床
  - に 助詞表示時間點
2. わたしは $\overset{きのう}{昨日}$ $\overset{べんきょう}{勉強}$しました。 我昨天有讀書
  - しました 過去式
	||肯定|否定|
	|--|--|--|
	|現在式|ます|ません|
	|過去式|ました|ませんでした|
3. 休み は 土曜日 と 日曜日 です。 休息日是星期六和星期日
  - と 助詞表示並列關係 (和)
4. 昼休み は 12時から 1時まで です。 午休是從12點到1點
	- から 助詞表示起點(從)
	- まで 助詞表示終點(到)

#### Lesson 5
1. わたし は 京都 へ 行きます 我去京都
	- へ 助詞表示方向(往) 助詞讀作え
2. わたしは タクシーで うちへ 帰ります 我搭計程車回家
	- で 助詞表示交通工具
3. わたしは 家族と 日本へ 来ました 我和家人來日本
	- と 助詞表示一起(和)
	- わたしは 一人で 学校へ 来ました 我一個人來學校
		- で 助詞表示動作進行單位
4. わたしは 7月15日に 国へ 帰ります 我7月15日回國
	- に 助詞表示時間點
5. どこ へも 行きませんでした 我哪裡都沒去
	- も 也/都 全否定
	
#### Lesson 6
1. わたしは 本を 読みます 我看書
	- を 助詞表示動作作用對象
	- わたしは 水を 飲みます 我喝水
2. わたしは 駅で 新聞を 買いました 我在車站買報紙
	- で 助詞表示動作發生地點
3. 一緒に 神戶へ 行きませんか 我們一起去神戶好嗎?
	- ません(否定型) + か 表示邀請
	- いっしょに 映画を 見ませんか 我們一起去看電影好嗎
    - ええ、見るしょう
		- すみません、ちょっと...
4. ちょっと 休みましょう 休息一下吧
  - ましょう 吧
	- 帰りましょう
5. うちで 宿題を します。それから、CDを 聞きます。我在家做作業。然後聽CD
	- それから 然後
  - ときどき 有時候
6. 友達 に 会います 我見朋友
	- に 助詞表示間接對象/接觸點
7. 何
  - 有兩種唸法なん / なに
		- た行,だ行,な行 唸なん
		- 其他唸なに

#### Lesson 7
1. わたしは パソコンで 映画を 見ます 我用電腦看電影
	- で 助詞表示手段/方法/工具
	- これわ 日本語で 何ですか 這個用日語怎麼說?
2. わたしは 木村さんに 花を あげます 我送花給木村先生
	- に 助詞表示動作的對象
3. わたしは カリナさんに チョコレートを もらいました 我從卡莉娜那裡得到了巧克力
  - 這裡に 也可以用 から(表示動作的接受方)
4. わたしは もう メールを 送りました 我已經寄出email了
	- もう 已經
	- もう 昼ご飯を 食べましたか 你已經吃午餐了嗎
		- はい、もう 食べました 我已經吃了
		- いいえ、まだです 我還沒吃

#### Lesson 8
1.  桜は きれい です。 櫻花很美(な形容詞)
2. 富士山は 高い です。 富士山很高(い形容詞)

	||肯定|否定|
	|--|--|--|
	|名詞|です|じゃありません|
	|な形容詞|です|じゃありません|
	|い形容詞|いです|くないです|

	- いいです / よいです 很好
	- よくないです 不好 (不好要用よいです變化)

3. 桜は きれいな 花です。 櫻花是美的花
  - な形容詞 + な + 名詞
4. 富士山は 高い 山です。 富士山是高的山
	- い形容詞 + 名詞
5. (日本の地下鉄)便利です。そして、きれいです。 方便。而且很漂亮
	- そして 接續詞 而且
6. (日本の車は) 高いですが、いいです。 雖然貴，但是很好
 - が 接續詞表示相反 但是
7. 奈良は どんな 町ですか 奈良是怎樣的城市?
	- どんな 什麼樣的
	- これは 何の本ですか 這是什麼書? (カメラの本です)
	  - 何の 有關什麼的

そして 和 それから 的差別
- そして 有連續性
- それから 無連續性

#### Lesson 9
1. わたしは イタリア料理が 好きです 我喜歡義大利料理
  - が 助詞表示焦點
2. わたしは 日本語が 少し わかります 我懂一點日語 (非動作動詞)
  - が 助詞表示焦點
  - わたしは 日本語を 勉強します (動作動詞)
	  - を 助詞表示動作作用對象
	- わたしは 時間が ありません 我沒有時間
3. 今日は 子供の 誕生日ですから、早く 帰ります 因為今天是小孩的
生日，所以我要早點回家
  - から 助詞表示原因
	- 疑問詞: 何(問事物)、誰(問人)、どこ(問地點)、いつ(問時間)、どうして
4. 我有~ の 的說法
  - 事情: 約束が あります 我有約
	- 東西: 車が あります 我有車 / 車を 持っています

 - おります(無生命) / います (有生命) 
3. 東京デイズニーランドは 千葉県に あります 東京迪士尼樂園在千葉縣
 - は 助詞表示主題

	 A: 古い手紙や 写真\[など\]が あります。 有舊信和照片
	 - や 助詞表示並列(和) (列舉不完的東西)

#### Lesson 11 數量詞
1. 会議室に テーブルが 7つ あります 會議室裡有7張桌子
 - つ 一般物品的數量詞
2. わたしは 日本に 1年 います 我在日本待一年
 - わたしは みかんを 3つ 食べました 我吃了3個橘子
 - 友達は シャツを 2$\overset{まい}{枚}$ 買いました 朋友買了2件襯衫
3. 1$\overset{しゅうかん}{週間}$に 1回 映画を 見ます 我一週看一次電影
 - に： 助詞表示頻率
- 1年に 5回ぐらい 映画を 見ます 我一年大約看5次電影
 - ぐらい 助詞表示概數量 大約 (くらい也可以)
4. Q. 何分 休みますか? 休息幾分鐘?
	 A. 10分だけ 休みます
	 - だけ 助詞表示限定 只有

#### Lesson 12
1. 昨日は 雨でした。 今天下雨了 
2. 昨日は 寒かったです。 昨天很冷。

* 動詞
	||肯定|否定|
	|--|--|--|
	|現在式|ます|ません|
	|過去式|ました|ませんでした|
* い形容詞
	||肯定|否定|
	|--|--|--|
	|現在式|いです|くないです|
	|過去式|かったです|くなかったです|
* な形容詞/名詞
	||肯定|否定|
	|--|--|--|
	|現在式|です|じゃありません|
	|過去式|でした|じゃありませんでした|

3. 北海道は 九州より 大きいです。 北海道比九州大
 - より 助詞表示比較對象
4. わたしは 1年で 夏が いちばん 好きです 我一年中最喜歡夏天
 - で 助詞表示範圍

  - どちら 哪一個 (兩選一)  
  - A: サッカーのほうが おもしろいです。 足球比較有趣
	- のほうが 助詞表示比較對象
	- どちらも 好きです 我兩個都喜歡

 - が 助詞表示焦點

#### Lesson 14
1. ちょっと 待って ください 請等一下
 - て形 + ください 請~
 - ドアを 開けます → ドアを 開けて ください 請開門
 - 日本語を 話します → 日本語を 話して ください
2. 荷物を 持ちましょうか 要不要幫你拿行李?
 - ~ます~ + ましょうか 要不要(為你)做~?
 - Q: ちょっと 休みましょうか
 - A: ちょっと 休みませんか 
3. ミラーさんは 今 電話を <u>かけて</u>います 米勒先生正在打電話
	- かけます → かけて (て形)
	- て形 + います 正在~
4. 喫茶店に $\overset{はい}{入}ります$
  - に 助詞表示進入點
5. 喫茶店を $\overset{で}{出}ます$
	- を 助詞表示離脫點
	- ご飯を 食べます (を動作對象)
	- 公園を 散歩します (を經過地點)
6. 雨が 降ります 下雨
  - が 助詞表示述語的主體
	- 空が 赤いです 天空是紅色的

#### Lesson 15
1. 写真を 撮っても いいですか? 我可以拍照嗎?
 - 撮ります → 撮って (て形)
 - て形 + も いいですか 可以~嗎?
2. サントスさんは 電子辞書を 持っています Santos先生有帶電子辭典
 - 持ちます → 持って (て形)
 - て形 + います 表示目前狀態
3. お酒を 飲んでは いけません 不能喝酒
 - 飲みます → 飲んで (て形)
 - て形 + は いけません 不能~
 - て形 + も いいです 可以做~
4. ここに 車を 止めては いけません 這裡不能停車
	- に 動作歸著點
5. 動作進行地點的で vs 動作歸著點的に
	- で：在場所做 ~ ここで車を止めてください 請在這裡停車
	- に：做在場所 ~ ここに車を止めてください 請把車停在這裡

#### Lesson 16
1. 朝 ジョギングを して シャワーを 浴びて 会社へ 行きます 早上慢跑、洗澡然後去公司
	- て形、て形、～ます  動作順序
2. コンサートが 終わってから、レストランで 食事しました 音樂會結束後，在餐廳用餐
	- て形から 做~之後
	- から： 助詞表示起點(從)、理由/因為、順序
3. 大坂は 食べ物が おいしいです 大阪的食物很好吃
	- が: 助詞表示焦點
4. この部屋は 広くて、明るいです 這個房間又寬敞又明亮
	- この部屋は 広いです。そして、明るいです
	- Aて形、B 既A又B
	- い形容詞て形變化: ~い~ → くて

#### Lesson 17
1. 写真を 撮らないで ください 請不要拍照
 - 撮ります → 撮らないで (ない形)
 - で 1.助詞表示樣態 2.助述詞だ的て形
 - ない形 + で ください 請不要做~
 - て形 + ください 請做~
2. パスポートを 見せなければ なりません 你必須出示護照
	- 見せます → 見せない(ない形) → 見せ~ない~ なければ なりません
	- なければ(不做~的話) なりません / いけません / ためです(不行) 
3. 日曜日は 早く 起きなくても いいです 星期日不必早起
	- 起きます → 起きない(ない形) → 起き~ない~ なくても いいです
	- なくても いいです 不做~也可以
4. レポートは あした 書きます 報告明天寫
	- は: 助詞表示區別

#### Lesson 18
1. ミラーさんは 漢字を 読むことが できます 米勒先生會讀漢字
	- 読みます → 読む (辭書形)
	- 辭書形ことが できます 會做~
	- 英語が できます 會說英語
	- 英語を 話すことが できます 會說英語
2. 私の趣味は映画を 見ることです 我的興趣是看電影
	- A は B辭書形ことです A是做B
3. 寝るまえに 日記を 書きます 睡覺前寫日記
	- 寝ます → 寝る (辭書形)
	- に 助詞表示時間點
	- A辭書形まえに、B。 做A之前做B
4. ここで 切符を 買う ことが できます 你可以在這裡買票
	- 買います → 買う (辭書形)
5. 趣味 vs 興味
	- 趣味: 已經開始做且是活動性的 趣味は～\[こと\]です。 (興趣是做～)
	- 興味: ～に興味があります (對～有興趣)

#### Lesson 19
1. 相撲を 見たことが あります 我看過相撲
	- 見ます → 見た (た形)
	- た形+ことが あります 有做過~
2. 休みの日は テニスをしたり、散歩に行ったりします
	- た形り、た形り します 做A做B (動作列舉)

#### Lesson 20
丁寧形(比較禮貌) vs 普通形(比較口語)

1. サントスさんは パーティーに こなかった Santos先生沒有來參加派對

|動詞|肯定|否定|
|--|--|--|
|現在式|来ます → 来る (辭書形)|来ません → 来ない (ない形)|
|過去式|来ました → 来た (た形)|来ませんでした → 来なかった (なかった形)|

2. 東京は人が多い。 東京人多

|い形容詞|肯定|否定|
|--|--|--|
|現在式|多いです → 多い|多くないです → 多くない|
|過去式|多かったです → 多かった|多くなかったです → 多くなかった|

3. 沖縄の海は きれいだった 沖繩的海很漂亮

|な形容詞/名詞|肯定|否定|
|--|--|--|
|現在式|きれいです → きれい\[だ\]|きれいじゃありません → きれいじゃない|
|過去式|きれいでした → きれいだった|きれいじゃありませんでした → きれいじゃなかった|

4. きょうは 僕の誕生日だ 今天是我的生日
	- 誕生日です → 誕生日\[だ\]

5. 丁寧體改為普通體
	- 一緒に コーヒーを ~飲みませんか~  飲まない?
	- 日本へ ~行きたいです~ 行きたい
	- ちょっと 待って~ください~
	- 今、晩ご飯を ~食べて~います~ いる
	- 早く帰っても ~いいです~ いい
	- 写真を 撮っては ~いけません~  いけない
	- たばこを 吸わないで ~ください~
	- レポートを 出さなければ ~なりません~ ならない

#### Lesson 21
1. わたしは あした 雨が 降ると 思(おも)います 我猜想明天會下雨
  - と： 助詞提示內容
	- 普通形(な形容詞/名詞 + だ)と 思います 猜想/認為~
	- 東京は 交通が 便利だと 思います 我認為東京交通方便
2. わたしは 父に 留学したいと 言いました 我跟我爸說我想要留學
	- に： 助詞表示對象
	- と： 助詞提示內容
	- 留学します → 留学したいです (想要) → 留学したい (普通體去掉です)
3. 疲れた でしょう？ 累了吧?
	- 疲れました → 疲れた (普通體去掉です)
	- 普通形でしょう？ (な形容詞/名詞不用加だ)
4. ちょっと ビール でも 飲みませんか？ 要不要喝點啤酒之類的?
	- でも 助詞表示舉例
	- コーヒー など いかがですか？ 要不要喝咖啡之類的?

## 文法
- 名詞 + します 可以變成動詞

## 助詞
### 格助詞
單字與動詞的關係，單字的意思是格助詞決定的
- で 地點
- から 表示起點(從)
- に 間接對象/時間
- を 對象/受詞
- が 主詞

### 係助詞/副助詞
- は (要唸成わ)
	- 表示句子主題(說明對象)
	- 表示對比
	- 表示特別提示
	- 加在助詞後面表示強調

### 助動詞
- ます/まし 表示禮貌
- ない 否定助動詞

### 終助詞
- か 表示疑問/反問/驚訝
