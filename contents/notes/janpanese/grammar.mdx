## 文法
- い形容詞 + 名詞
	- おいしい + パン 美味的麵包
	- やすい + ホテル 便宜的旅館
- な形容詞 + な + 名詞
	- にぎやか な まち 熱鬧的街道
	- ゆうめい な ばしょ 有名的地方(有名是い結尾但是是な形容詞)

形容詞(い形容詞)變化規則:
- 形容詞い + 名詞
- 形容詞 ~い~ + くない (否定)
- 形容詞~い~ + くて (中止型,還沒講完)
- 形容詞~い~ + かった (過去) 

形容動詞(な形容詞)變化
- + だ 原形
- + で 表示還沒講完(既...又...)
- + な 後面加名詞
- + に 後面加動詞

## 助詞
### 格助詞
- で 地點
- から 表示起點(從)
- に 間接對象/時間
- を 直接對象
- が 動作者

### 係助詞/副助詞
- は (要唸成わ)
	- 表示句子主題(說明對象)
	- 表示對比
	- 表示特別提示
	- 加在助詞後面表示強調

### 助動詞
- ます/まし 表示禮貌
- ない 否定助動詞

### 終助詞
- か 表示疑問/反問/驚訝
