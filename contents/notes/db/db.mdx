#### Q2. Employees Earning More Than Their Managers

```sql
SELECT E1.name as Employee
FROM Employee as E1
LEFT JOIN Employee as E2
ON E1.ManagerId = E2.Id

```

#### Assignment - Employees Whose Manager Left the Company

```sql
SELECT E1.employee_id
FROM Employees as E1
LEFT JOIN Employees as E2
ON E1.manager_id = E2.employee_id
WHERE E1.salary < 30000 AND E1.manager_id IS NOT NULL AND E2.employee_id IS NULL
ORDER BY E1.employee_id
```

#### Q3. Monthly Transactions I
- 把日期依照年月分組group by date_format(date, '%Y-%m')
- sum(state) / sum(state='approved') 可以加條件
- CASE WHEN ... THEN ... ELSE ... END

```sql
SELECT DATE_FORMAT(trans_date, '%Y-%m') AS month, 
	country,
	count(*) as trans_count,
	sum(state='approved') as approved_count,
	sum(amount) as trans_total_amount,
	sum(CASE WHEN state='approved' THEN amount ELSE 0 END) as approved_total_amount
FROM Transactions
GROUP BY DATE_FORMAT(trans_date, '%Y-%m'), country
```

#### Q4. User Activity for the Past 30 Days I
統計一個月內的活躍用戶(distinct / between and)

```sql
select 
    activity_date as day,
    count(distinct user_id) as active_users 
from Activity
where activity_date between '2019-06-28' and '2019-07-27'
group by activity_date
```

#### Assignment - List the Products Ordered in a Period
過濾日期、合併兩張表、group by product_name、統計數量並過濾

```sql
select p.product_name, sum(unit) as unit
from Orders as o
left join Products as p
on o.product_id = p.product_id
where o.order_date between '2020-02-01' and '2020-02-29'
group by product_name
having sum(unit) >= 100
```

## Grouping & Join Aggregation Library
#### Q1. Students and Examinations
CROSS JOIN: 因為沒有參加考試也要顯示出科目是0次，學生要先和科目CROSS JOIN

```sql
SELECT s.student_id, s.student_name, sub.subject_name, count(e.subject_name) as attended_exams
FROM Students s
CROSS JOIN Subjects sub
LEFT JOIN Examinations e
ON (s.student_id = e.student_id) AND (sub.subject_name = e.subject_name)
GROUP BY s.student_id, s.student_name, sub.subject_name
ORDER BY s.student_id
```

Q2. Customers Who Bought All Products
查詢哪些消費者購買全部的商品，消費紀錄可能會有同個消費者買同一個商品多次
(SUBQUERY, DISTINCT)

```sql
SELECT customer_id
FROM Customer 
GROUP BY customer_id
HAVING COUNT(DISTINCT product_key) = (
    SELECT COUNT(*)
    FROM Product
)
ORDER BY customer_id  
```

Q3. Tree Node
判斷樹的節點是Root,Inner還是Leaf。(CASE WHEN THEN END / subquery)

```sql
SELECT
    id,
    CASE
        WHEN p_id IS NULL THEN 'Root'
        WHEN id IN (SELECT p_id FROM Tree WHERE p_id IS NOT NULL)
            THEN 'Inner'
        ELSE 'Leaf'
    END AS type
FROM Tree;
```

Q4. Movie Rating
找出投最多票的user name和rating最高的movie title最後合併結果
(UNION ALL / GROUP / ORDER BY / JOIN)

```sql
(SELECT name AS results
FROM MovieRating r
JOIN Users u ON r.user_id = u.user_id
GROUP BY r.user_id
ORDER BY count(rating) DESC, u.name ASC
LIMIT 1)
UNION ALL
(SELECT m.title as results
FROM MovieRating r
JOIN Movies m
    ON m.movie_id = r.movie_id
WHERE r.created_at >= '2020-02-01'
  AND r.created_at < '2020-03-01'
GROUP BY r.movie_id
ORDER BY AVG(rating)  DESC, m.title ASC
LIMIT 1)
```
