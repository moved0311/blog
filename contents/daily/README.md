# Daily Notes

This directory is for private daily learning logs.

Files in this directory are intentionally ignored by the blog content loader, so they will not appear on the homepage, the notes page, or any existing post route.

Suggested filename format:

```text
YYYY-MM-DD.md
```

Suggested entry format:

```markdown
# YYYY-MM-DD

## Learned

- 

## Questions

- 

## Next

- 
```
