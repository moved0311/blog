## CS50 SQL - Lecture 4 - Viewing

如果不想要永久把VIEW存起來 -> TEMPORARY VIEW
有效期限是連線DB時，重新連線時就消失

刪除VIEW: DROP VIEW

CTE(Common Table Expression)

```sql
WITH name AS (
	SELECT ...
), ...
SELECT ... FROM name;
```

Securing
可以根據權限拆分VIEW，讓不同權限的人可以讀取不同VIEW

Trigger
不能直接刪除/插入View，但是可以透過Trigger的INSTEAD OF操作原始TABLE
達到效果。
