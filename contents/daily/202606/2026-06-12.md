STOREAGE CLASS:

Type Affinites:
INTEGER, TEXT, REAL, NUMERIC, BLOB
SQLite特別的設計，一般資料庫如果插入的欄位和定義的不同會報錯，但SQLite會幫你轉型

Affinity vs Storage Class
| 項目          | 意義               |
| ------------- | ------------------ |
| Type Affinity | 欄位偏好的型別     |
| Storage Class | 資料實際儲存的型別 |

SQLite 的 Type Affinity 不是強制型別，而是「欄位希望資料長成什麼型別」的規則。SQLite 會嘗試自動轉換，但不一定成功，也不一定拒絕
