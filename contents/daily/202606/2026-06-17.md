Session: Session 是伺服器端儲存的狀態
	伺服器通常會：
	建立 Session 資料
	產生 Session ID
	把 Session ID 放進 Cookie

Cookie: 發送request到url時會自動帶入，例如訪問 /api/refresh 會自動帶上 refreshtoken。 

WebStorage: 可以暫存登入拿到的token。
LocalStorage: 永久保存，同網域都能讀，關閉瀏覽器不會消失
SessionStorage: 關閉Tab消失，不同Tab不共享

Refresh Token 放在HttpOnly cookie，比較安全js無法存取

Process
Process是作業系統分配資源的單位
有自己的Memory, Head, Stack, File Descriptor
Process 之間預設不能直接存取彼此記憶體。

Thread
Thread 是 CPU 排程的單位。Process可以有多個Thread
Thread共用Heap, Global Variables, Open Files
Thread有自己的Stack, Program Counter

Python併發寫法
```python
# 沒併發：每次 await 都等前一個完成
for job_id in range(1, count + 1):
    results.append(await async_job(job_id, delay))

# 有併發：先建立多個 coroutine，再一起 await
tasks = [async_job(job_id, delay) for job_id in range(1, count + 1)]
results = await asyncio.gather(*tasks)
```
