FOREIGN KEY CONSTRAINT
如果PRIMARY KEY有被其他TABLE參照，不能直接刪除PRIMARY KEY

或是設定FOREING KEY刪除條件
FOREIGN KEY('artist_id') REFERENCE artist(id)
ON DELETE CASCADE/SET NULL


```sql
UPDATE "votes" SET "title" = trim("title");

```

TRIGGER
當Table有變動時，可以建立一個Trigger自動做一些SQL操作

```sql
CREATE TRIGGER name
AFTER/BEFORE INSERT/UPDATE/DELETE ON table
FOR EACH ROW
BEGIN
   ...;
END;
```

```sql
CREATE TRIGGER "sell"
BEFORE/INSTEAD OF/AFTER INSERT/DELETE ON "collections"
FOR EACH ROW WHEN condition
BEGIN
    INSERT INTO "transactions" ("title", "action")
		VALUES (OLD."title", 'sold'); 
END;
-- DELETE搭配OLD 剛剛刪除的
-- INSERT搭配NEW 剛剛加入的
```

Soft Deletions

